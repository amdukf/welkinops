# Vagrant & UTM in MacOS (Apple silicon)

## Overview

This documentation covers the process of creating a custom Ubuntu Server 24.04 virtual machine using [UTM](https://mac.getutm.app), packaging it as a Vagrant `.box`, and using it with the [vagrant-utm](https://github.com/naveenrajm7/vagrant_utm) provider. It also documents key fixes and requirements for SSH, guest agents, and networking.
